产品分类 JSON 接口

下面是三个可直接访问的动态 JSON 接口链接。二维码可以直接指向这些地址，扫码后会返回对应母类下随机挑选的一个子类。

返回值已经简化为最小结构，例如 {"items":"苹果"}。 每次请求 /api/food.json、/api/daily.json、/api/electronics.json 时，都会返回一个新的随机结果。

食品

/api/food.json

每次访问会从苹果、香蕉、橙子、猪肉、牛肉、鸡肉、牛奶、酸奶等多个子类里随机返回一个。

日用品

/api/daily.json

每次访问会从牙刷、牙膏、毛巾、洗衣液、洗洁精、香皂、纸巾等多个子类里随机返回一个。

电子产品

/api/electronics.json

每次访问会从手机、耳机、键盘、鼠标、显示器、平板电脑、路由器等多个子类里随机返回一个。
