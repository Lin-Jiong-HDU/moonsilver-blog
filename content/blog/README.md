# Blog Content

把博客文章放在这个文件夹里即可。每篇文章一个 `.txt`、`.md` 或 `.markdown` 文件，文件名就是文章的 slug。

最简单的写法：

```txt
文章标题

这里写第一段正文。没有额外配置时，第一行会作为标题，后面的第一段会作为摘要。

继续写正文即可。
```

如果需要日期、标签或自定义摘要，可以在文件开头加 frontmatter：

```md
---
title: 文章标题
excerpt: 一段简介，会显示在列表页。
publishedAt: 2026-04-23
updatedAt: 2026-04-23
tags: 博客, 随笔
---

正文内容。
```

建议：
- 文件名尽量用英文小写和连字符，比如 `local-first-blog.txt`
- 正文可以直接写普通文本，也可以使用简单 Markdown 标题、列表、引用、链接
- 新增、修改后直接提交并推送，站点会在构建时读取这里的内容
